# GuildPass Integrations

GuildPass helps Web3 communities manage token-gated membership, roles, and access decisions from a core API while keeping integrations lightweight. This documentation site introduces the integration starter, the dashboard-facing SDK utilities, and the webhook helpers used to connect Discord bots, dashboards, and future community tools to the GuildPass ecosystem.

## Start Here

- Read the [Product Overview](/docs/overview)
- Follow the [Admin Quickstart](/docs/admin-quickstart)
- Build locally with the [Developer Quickstart](/docs/dev-quickstart)
- Review the [API Overview](/docs/api-overview)

## SDK And Integration Docs

- Explore this repo's [`integration-client`](https://github.com/Adamantine-guild/guildpass-app/tree/main/packages/integration-client) package for typed calls into GuildPass core APIs
- Use [`webhook-utils`](https://github.com/Adamantine-guild/guildpass-app/tree/main/packages/webhook-utils) for webhook verification helpers and examples
- Browse the dedicated [`guildpass-sdk`](https://github.com/Adamantine-guild/guildpass-sdk) repository for the official TypeScript SDK

## GitHub Repositories

- [`guildpass-core`](https://github.com/Adamantine-guild/guildpass-core): core contracts, access API, and policy engine
- [`guildpass-app`](https://github.com/Adamantine-guild/guildpass-app): web app, dashboard, and package workspace
- [`guildpass-integrations`](https://github.com/Adamantine-guild/guildpass-integrations): Discord bot, docs, and ecosystem integration starter
